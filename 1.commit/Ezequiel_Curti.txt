Teorica:

.1.1) Para iniciar un repositorio local se debe ejecutar el comando "Git Init".

1.1.2) Para que un directorio deje de ser controlado por git debemos usar el comando "git rm --r <Directorio>".

1.1.3) Si, una vez que un directorio está controlado por git, todo archivo creado dentro de este tambien estara controlado por el repositorio Git.

1.1.4) Para agregar un archivo a un repositorio local de git hay que colocar el comando "Git add <Archivo>".

1.1.5) El comando que se debe usar para determinar el estado de los archivos es "Git status".

1.1.6) Para realizar el commit de un archivo hay que colocar "Git commit" y luego el nombre del archivo del que se desea realizar el commit.

1.1.7) Un commit en Git es como una captura que realiza a un archivo determinado en un momento determinado para guardarlo y subirlo al repositorio, algo asi como una copia de seguridad que debe estar acompañada de un mensaje para guardarse.
